remoting存储类型 #

连接模式为server模式时就是用的这种存储类型。
使用这种存储类型，客户端不在直接存储和任务的相关操作，简化了tcc配置同时减轻系统负担。

如何使用 #

选择注册中心 #

查看注册中心说明

server部署 #

查看server部署说明

添加配置 #

spring:
  tcc:
    storage:
      storage-type: remoting
    registry:
      registry-type: direct
      cluster-name: default
      direct:
        server-addresses: 127.0.0.1:2332
      zookeeper:
        connect-string: 127.0.0.1:2181
        max-retries: 4
      nacos:
        server-addr: 127.0.0.1:8848

免责声明

TCC-TRANSACTION是一款开源的微服务架构下的TCC型分布式事务解决方案，致力于提供高性能和简单易用的分布式事务服务。
